Spacing

Intro paragraph about uniform spacing between elements. Our base spacing unit is 16px / 1rem which is the default browser text size. Spacing for margin and padding is based on increments of our $spacer variable: 0px, 16px / 1rem, 32px / 2rem, 64px / 4rem, 96px / 6rem, 128px / 8rem.

$spacers: (
	0: 0,
	1: $spacer, // 16px
	2: $spacer * 2, // 32px
	3: $spacer * 4, // 64px¬†
	4: $spacer * 6, // 96px
	5: $spacer * 8 // 128px
) !default;

Grid System

The bootstray grid system is based on 12 columns. Grid columns can occupy ifferent widths at different viewport sizes. The class names have the following convention:

<div class="col-breakpointsize-columnwidth">

Example: A column that is 50% width for desktop and 100% width for mobile: 

<div class="col-lg-6 col-sm-12">
